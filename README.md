# Reservoir-Sampling-for-Stream-Data

TOPIC 14: Reservoir Sampling for Stream Data
Concepts: Reservoir Sampling, Streaming Algorithms
Create a system that maintains a random sample of 100 items from a continuous stream of 100,000
items using reservoir sampling. Show that the sample is representative by comparing sample
statistics (mean, distribution) with the full stream.
Dataset: Generate synthetic stream of numbers (normal distribution, random values)
Success Criteria: Sample statistics closely match population statistics (within 5%)

Reservoir Sampling guarantees fairness in one pass, constant memory
